Grafana for Windows — MSI installer & quick guides

Download and install Grafana on Windows (x64), run it as a Windows Service, collect Windows logs with Promtail, and learn common troubleshooting steps. Clear, concise guides for admins and devops.

Quick start

  1. Click the download button (official MSI). The file is signed by Grafana Labs.
  2. Run the MSI and follow the installer prompts.
  3. To run as a service, follow the "Install as Service" guide linked in the menu.
  4. For logs and Promtail configuration see the Promtail guide.

System requirements & default ports

  • OS: Windows 10/11 or Windows Server 2016+ (x64).
  • Resources: at least 1 CPU core and 1 GB RAM for small deployments (more recommended for plugins and dashboards).
  • Default HTTP port: 3000. Changeable in configuration.
  • Firewall: allow inbound on your chosen HTTP/HTTPS port for intended LAN/WAN ranges only.

Install as a Windows Service (short version)

  1. Install Grafana via MSI.
  2. Open the Windows Services console and locate the Grafana service.
  3. Set Startup type to Automatic, start the service.
  4. Open http://localhost:3000 to verify Grafana is running.
  5. Secure the admin account and restrict firewall scope to needed networks.
Full details in Install as Service.

Promtail on Windows (quick overview)

  1. Install Promtail for Windows.
  2. Configure scraping for Windows Event Logs and selected log files.
  3. Point Promtail to your Loki endpoint and verify ingestion.
  4. Create dashboards in Grafana to explore logs and alerts.
See Promtail on Windows for configuration patterns.

Upgrade & rollback (safe approach)

  1. Read the release notes for breaking changes.
  2. Back up the data directory and configuration.
  3. Run the new MSI and follow the prompts.
  4. If issues arise, stop the service, restore backup and consider previous MSI.
More in Upgrade & Rollback.

Verify installer integrity & signature

  1. Download the MSI from the official site.
  2. Check the digital signature in file properties (Publisher: Grafana Labs).
  3. Optionally, compare checksums provided by the vendor.

Frequently asked questions

Is Grafana for Windows free?
Yes, the OSS MSI is free to download and use.

Does the MSI install Grafana as a Windows service?
The MSI installs Grafana and can run it as a Windows service. Manage it via Services or PowerShell.

Which Windows versions are supported?
Windows 10/11 and Windows Server 2016+ (x64).

What is the default port?
3000. You can change the port and update firewall rules accordingly.

Where are data and logs stored?
Under the Grafana data directory. Paths can be customized in configuration.

How do I uninstall?
Use Apps & Features. Stop the service first if needed; remove custom firewall rules.

Can I run behind IIS?
Yes—configure reverse proxy (HTTPS, headers) and adjust Grafana settings.

Is Local System safe?
Prefer a dedicated, least‑privilege service account when possible.
