Download Grafana for Windows (x64 MSI)

Get the official Grafana MSI for Windows. This page helps you choose the right build, verify integrity, and start safely on Windows 10/11 and Windows Server 2016+ (x64).

Choose the right build

  • Grafana OSS (recommended here): free, core features, community plugins.
  • Grafana Enterprise: enterprise capabilities and plugins; available on the official downloads page.
We link to Grafana's official downloads. Always verify the publisher signature (Grafana Labs) after download.

Verify installer integrity

  1. Right‑click the downloaded MSI → Properties → Digital Signatures.
  2. Ensure the signature lists Grafana Labs as the Publisher and the signature is valid.
  3. Optionally compare checksums published by Grafana.

System requirements

  • OS: Windows 10/11, Windows Server 2016+ (x64).
  • Admin rights needed to install/run as a Windows Service.
  • Default port: 3000 (adjust Windows Firewall scope as needed).

After download

  • Run the MSI and follow the installer prompts.
  • Open Services and ensure Grafana service is running (if installed as service).
  • Browse to http://localhost:3000 and complete setup.
